# Receipt-Image-Processing-for-OCR

        การจดจำตัวอักษรด้วยแสง (OCR) จากภาพถ่ายใบเสร็จในสภาพแวดล้อมจริงเป็นปัญหาที่ท้าทาย เนื่องจากภาพที่ได้มักมีคุณภาพต่ำจากหลายปัจจัย เช่น แสงสว่างไม่สม่ำเสมอ รอยพับบนกระดาษ และลายน้ำโลโก้และข้อความด้านหลังใบเสร็จส่องผ่านกระดาษบางมาปรากฏซ้อนทับบนเนื้อหาหลัก งานวิจัยนี้จึงมีวัตถุประสงค์เพื่อพัฒนาและประเมินประสิทธิภาพของกระบวนการประมวลผลภาพดิจิทัล สำหรับเตรียมภาพถ่ายใบเสร็จให้มีคุณภาพเหมาะสมก่อนส่งเข้าระบบ OCR

        กระบวนการที่พัฒนาขึ้นประกอบด้วย 2 ขั้นตอนหลัก ได้แก่ การแยกใบเสร็จออกจากพื้นหลังโดยใช้ Otsu's Thresholding และ Morphological Closing และการปรับปรุงคุณภาพภาพโดยใช้ Background Division เพื่อแก้ไขแสงเงาที่ไม่สม่ำเสมอ ร่วมกับ Wavelet Sharpening เพื่อเพิ่มความคมชัดของตัวอักษร และ CLAHE เพื่อปรับ Contrast เฉพาะพื้นที่

        ผลลัพธ์ที่ได้แสดงให้เห็นว่ากระบวนการนี้สามารถตัดพื้นหลัง แก้ไขความไม่สม่ำเสมอของแสง และเพิ่มความคมชัดของตัวอักษรได้อย่างมีประสิทธิภาพ อย่างไรก็ตาม ปัญหาจากลายน้ำโลโก้สีน้ำเงินและข้อความกลับหัวจากด้านหลังใบเสร็จยังคงปรากฏในภาพผลลัพธ์ เนื่องจากการประมวลผลในรูปแบบ Grayscale ไม่สามารถแยกแยะความแตกต่างของสีระหว่างตัวอักษรสีดำและลายน้ำสีน้ำเงินได้ งานวิจัยนี้ชี้ให้เห็นว่าการประมวลผลภาพสี (Color Image Processing) ร่วมกับการกรอง Channel สีโดยเฉพาะ หรือการนำเทคนิค Deep Learning มาเสริม อาจเป็นแนวทางที่จำเป็นสำหรับการแก้ไขปัญหาในเชิงลึกต่อไปในอนาคต
